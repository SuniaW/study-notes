

# 免费 Markdown 编辑器推荐及 VS Code 使用指南

本文档旨在为您推荐市面上主流的免费 Markdown 编辑器，并提供在 Visual Studio Code (VS Code) 中创建和编辑 Markdown 文件的详细指南。

## 第一部分：免费 Markdown 编辑器推荐

根据不同的**使用习惯**（“所见即所得”或“双栏预览”）以及**使用场景**（单纯写作、笔记管理、代码编写），以下是几款最佳的免费 Markdown 编辑器：

### 1. 最佳“所见即所得”编辑器：MarkText
如果你喜欢 Typora 那种简洁、写完代码直接渲染成样式的“所见即所得”模式，MarkText 是目前最好的开源免费替代品。
* **优点：** 完全免费开源、所见即所得（即打即看，没有分割线）、界面极其简洁、支持多种主题。
* **缺点：** 启动速度稍慢，更新频率不如商业软件高。
* **适用平台：** Windows, macOS, Linux

### 2. 最佳“知识库/笔记管理”工具：Obsidian (黑曜石)
目前最火的本地知识库工具，被誉为个人的“第二大脑”。虽然有付费同步服务，但软件本地功能对个人完全免费。
* **优点：** 本地存储（数据安全）、插件生态极其丰富、支持双向链接和白板（Canvas）功能。
* **缺点：** 上手门槛稍高，需要花时间折腾插件。
* **适用平台：** Windows, macOS, Linux, iOS, Android

### 3. 最佳“跨平台同步”笔记：Joplin
一款主打“笔记管理”和“多端同步”的开源加密笔记应用。
* **优点：** 开源免费、支持端到端加密、支持 WebDAV/OneDrive 等多种网盘免费同步、配备好用的网页剪藏插件。
* **缺点：** 默认界面较朴素，采用传统的“左侧编辑、右侧预览”模式。
* **适用平台：** 全平台（含移动端）

### 4. 程序员与技术人员首选：Visual Studio Code (VS Code)
通过配置插件，这款代码编辑器可以变身为宇宙最强 Markdown 编辑器之一。
* **优点：** 极其强大且完全免费。支持 Git 版本控制，高度定制化。
* **缺点：** 对非技术人员来说，界面略显复杂。
* **适用平台：** Windows, macOS, Linux

### 5. 学术写作与深度研究：Zettlr
专为学术写作、研究人员设计的 Markdown 编辑器。
* **优点：** 深度集成 Zotero（文献管理），支持卡片盒笔记法（Zettelkasten），支持导出复杂的 PDF/Word 格式。
* **缺点：** 功能偏向学术圈，普通用户可能觉得部分功能冗余。
* **适用平台：** Windows, macOS, Linux

### 6. 最佳免安装在线编辑器：StackEdit
如果你不想安装软件，只需在浏览器里快速写作。
* **优点：** 浏览器打开即用、支持离线写作、可直接同步到 Google Drive 或 GitHub。
* **缺点：** 依赖浏览器缓存，不适合管理大量本地文件。

**💡 快速选型建议：**
* **0 干扰沉浸式写作：** 选 **MarkText**
* **建立个人知识体系：** 选 **Obsidian**
* **免费多端同步笔记：** 选 **Joplin**
* **技术文档编写：** 选 **VS Code**

## 第二部分：VS Code 中创建与编辑 Markdown 指南

*注：Markdown 文件的标准后缀名为 **`.md`**（`.mk` 通常是指 Makefile 编译文件，请注意区分）。*

### 步骤一：创建 `.md` 文件
在 VS Code 中创建文件主要有两种常用方法：

**方法 A：快捷键新建（快速）**
1. 打开 VS Code。
2. 按快捷键 `Ctrl + N` (Win/Linux) 或 `Cmd + N` (macOS) 新建空文件。
3. 按 `Ctrl + S` (Win/Linux) 或 `Cmd + S` (macOS) 保存。
4. 在保存对话框中，将文件命名为 `你的文件名.md`。

**方法 B：在资源管理器中新建（推荐）**
1. 在左侧的文件栏（资源管理器）空白处右键单击。
2. 选择 **"New File" (新建文件)**。
3. 输入完整文件名（如 `readme.md`），回车即可。

### 步骤二：实时预览文件
Markdown 是一种标记语言，写的是带有排版语法的源码，看的是渲染后的效果。VS Code 提供了便捷的预览功能：

* **侧边栏双屏预览（一边写一边看）：**
    * 快捷键：按住 `Ctrl + K` (Mac 为 `Cmd + K`)，松开后紧接着按 `V`。
    * 鼠标操作：点击右上角的“放大镜+分屏”图标 (Open Preview to the Side)。
* **全屏预览：** 按 `Ctrl + Shift + V` (Win) / `Cmd + Shift + V` (Mac)。

### 步骤三：安装必备插件（提升体验）
虽然 VS Code 原生支持 Markdown，但安装插件可以解锁快捷排版、自动生成目录等高阶功能。

1. 点击左侧活动栏的 **"Extensions" (扩展)** 图标（或按 `Ctrl+Shift+X`）。
2. 在搜索框输入 **`Markdown All in One`**。
3. 点击 **Install (安装)**。

**安装后的常用快捷键技巧：**
* **加粗文字：** 选中文本后按 `Ctrl + B`
* **斜体文字：** 选中文本后按 `Ctrl + I`
* **自动生成目录 (TOC)：** 按 `Ctrl+Shift+P` 打开命令面板 -> 输入 `Create Table of Contents` 并回车。
