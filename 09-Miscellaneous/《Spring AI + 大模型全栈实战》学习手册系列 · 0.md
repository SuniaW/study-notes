# 《Spring AI + 大模型全栈实战》学习手册系列 · 前言

---

## 📖 写在前面

2024 年以来，大模型技术以惊人的速度重塑着软件开发的边界。从 ChatGPT 的爆火到各类开源模型的百花齐放，AI 不再是实验室里的概念，而是成为了每个开发者都需要掌握的核心技能。

**作为一名开发者**，和很多人一样，面对这场技术变革，既兴奋又焦虑：

- 兴奋的是，我们终于可以用自然语言与程序对话，构建前所未有的智能应用
- 焦虑的是，技术迭代太快，今天学的明天可能就过时了

于是，我决定**不再等待，动手实践**。

---

## 🎯 为什么写这套学习手册

在摸索 Spring AI + 大模型开发的过程中，我遇到了很多开发者共同的困惑：


| 困惑         | 现实                   |
| ------------ | ---------------------- |
| 教程太多太散 | 缺乏系统化的学习路径   |
| 示例过于简单 | 无法直接应用到生产环境 |
| 只讲调用 API | 缺少架构设计和工程实践 |
| 忽略运维部署 | 上线后问题频发无从下手 |

**这套学习手册，就是我为自己、也为同样在路上的你，整理的一份「从入门到生产」的实战指南。**

## 💡 学习建议

1. **按顺序阅读** — 专题之间有递进关系，建议循序渐进
2. **动手实践** — 代码一定要自己跑一遍，理解才会深刻
3. **结合实际** — 将示例适配到你的业务场景，效果更佳
4. **持续更新** — AI 技术迭代快，保持学习和关注
